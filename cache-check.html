🛠️ 强制刷新方法

Windows / Linux:

  • 硬刷新: Ctrl + Shift + R 或 Ctrl + F5
  • 清除缓存: Ctrl + Shift + Delete

Mac:

  • 硬刷新: Cmd + Shift + R
  • 清除缓存: Cmd + Option + E (开发者工具中)

开发者工具方法:

  1. 按 F12 打开开发者工具
  2. 进入 Network 标签页
  3. 勾选 "Disable cache"
  4. 刷新页面

📋 技术说明

期望版本: v2.0.1

旧版本: v1.0.4 或无版本参数

缓存原因: 浏览器保存了旧版本的CSS和JS文件

解决方案: 强制刷新或清除缓存
